Bracknell Forest Code Club

The home for Code Clubs run in the network of Bracknell Forest Libraries

How Code Club Works

Our enthuastic volunteers run free weekly code clubs at libraries across the Bracknell Forest area, for children aged 9-11.

Code Clubs give all children the opportunity to learn to code, no matter who they are or where they come from.

Each week our voluteers set a coding puzzle for our children to complete. The volunteers are on hand to help out when the children run into problems.

Projects are completed in a number of different technologies, including Scratch, HTML and Python.

The video below is provided by Code Club, and gives you an idea of how the club operates.

Projects

The children work on small projects, starting with Scratch in the first two terms, followed by two terms of HTML and CSS, and finishing with Python.

More details about the projects the children have completed can be found in our Projects pages
